# Blog Application

## Setup

### Backend
1. `cd backend`
2. `npm install`
3. Create `.env` (see `.env.example`)
4. `npm run dev`

### Frontend
1. `cd blog-frontend`
2. `npm install`
3. `npm start`

## Deployment

- Backend: [your-backend-url]
- Frontend: [your-frontend-url]

## Features

- User authentication (JWT)
- Blog CRUD (only author can edit/delete)
- Public blog listing with pagination
- Responsive design
